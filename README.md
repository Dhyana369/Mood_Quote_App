# ✨ Mood Quote Generator

A simple web application that generates motivational quotes based on your mood.

Built using **Python**, **Streamlit**, and **Pandas**.

## 📌 Features

* Select your mood from a dropdown
* Get a random motivational quote
* Clean and interactive UI
* Mood-based quote filtering
* Mood distribution visualization

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
